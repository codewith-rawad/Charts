Line Chart

A line chart is useful for showing trends over time. It displays information as a series of data points called 'markers' connected by straight line segments.

Pie Chart

A pie chart is used to represent data in a circular graph, where each slice represents a category's contribution to the whole dataset.

Bar Chart

Bar charts are used to compare different categories of data. The length of each bar is proportional to the value of the category.

Scatter Chart

A scatter chart displays values as a collection of points, showing relationships between variables and helping to identify correlations.

Area Chart

An area chart is similar to a line chart, but the area below the line is filled with color. It is used to represent quantities over time or categories.
